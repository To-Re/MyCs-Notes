# 1 数据流重定向

| 文件描述符 | 设备        | 说明     |
| ---------- | ----------- | -------- |
| 0          | /dev/stdin  | 标准输入 |
| 1          | /dev/stdout | 标准输出 |
| 2          | /dev/stderr | 标准错误 |



`>` 覆盖重定向

`>>` 追加重定向



**下面配合文件描述符例子**

`2>` 标准错误重定向

`1>` 标准输出重定向

`1>1.txt 2>&1` 标准错误和标准输出都输出至 `1.txt`
